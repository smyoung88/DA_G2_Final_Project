United States Traffic Accidents
Between 2016 and 2020

Using Machine Learning to predict the severity of traffic accidents.

Using a dataset provided by Sobhan Moosavi on Kaggle containing information on almost 3 million incidents, a machine learning model was generated in which the severity of a traffic accident was predicted based on almost four dozen factors.  By splitting the data into training and test data, creating a model where the accident severity was the target for the training data and removing the accident severity from the test data, an ML model was created which is able to precict with 85% accuracy what the severity of an accident would be based on the factors provided.  Moosavi defines the severity of the accident as: "a number between 1 and 4, where 1 indicates the least impact on traffic (i.e., short delay as a result of the accident) and 4 indicates a significant impact on traffic (i.e., long delay)".
